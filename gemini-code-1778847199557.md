# StudyOS Pro — A Level Command Center

StudyOS Pro is a high-performance, futuristic study dashboard specifically designed for Cambridge International AS & A Level students. It combines AI-powered learning tools with productivity tracking and exam management in a sleek, glassmorphic UI.

## 🚀 Features

* **AI Study Suite:** Integrated with the HuggingFace API to generate instant notes, quizzes, and detailed explanations for complex A-Level topics.
* **Exam Command Center:** Real-time countdowns and progress tracking for upcoming exam dates to stay ahead of the curve.
* **Productivity Tools:** * Built-in **Pomodoro Timer** with deep-focus tracking.
    * **Daily Progress Tracker** to log study hours and tasks.
    * Integrated **Study Notes** area that saves automatically.
* **Active Recall Flashcards:** Create and review custom flashcards with smooth flip animations to master key definitions.
* **Syllabus Navigator:** Quick-access tabs for core subjects (Computer Science, Mathematics, Physics) with direct links to past papers and top-tier educational resources like Papersdock.
* **Cyberpunk Aesthetics:** A "Midnight Neon" UI featuring glassmorphism, Inter and Orbitron typography, and responsive design.

## 🛠️ Technical Stack

* **Frontend:** HTML5, CSS3 (Custom Glassmorphism Design), Vanilla JavaScript.
* **APIs:** HuggingFace Inference API for AI-powered content generation.
* **Icons:** FontAwesome 6.5.0.
* **Fonts:** Inter, Orbitron, and JetBrains Mono.
* **Storage:** LocalStorage for persistent data (No account required).

## 📦 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/study-os-pro.git](https://github.com/yourusername/study-os-pro.git)
    ```
2.  **Open the project:**
    Simply open `index.html` in your favorite modern browser.

3.  **Activate AI Features:**
    * Go to [HuggingFace Settings](https://huggingface.co/settings/tokens).
    * Create a "Read" token.
    * Enter the token into the API banner in StudyOS Pro to enable the AI tools.
